┌──────────────────────────────────────────────────────────────────────────────┐
│                      🔨 PASOS PARA INSTALACIÓN                               │
└──────────────────────────────────────────────────────────────────────────────┘

FASE 1: PREPARACIÓN
  [ ] 1. Leer ASSEMBLY_GUIDE.md completo
  [ ] 2. Leer BOM.md y anotar componentes necesarios
  [ ] 3. Comprar todos los componentes
  [ ] 4. Reunir herramientas (soldador, multímetro, etc)

FASE 2: ENSAMBLAJE EN BANCO
  [ ] 5. Verificar voltaje step-down (5.0V ± 0.2V)
  [ ] 6. Ensamblar sensor NTC + resistencia 10K
  [ ] 7. Conectar LCD I2C
  [ ] 8. Conectar botones PAGE y PEAK
  [ ] 9. Verificar todas las conexiones (QUICK_REFERENCE.txt)

FASE 3: PROGRAMACIÓN
  [ ] 10. Abrir ms2-display.ino en Arduino IDE
  [ ] 11. Verificar config.h: TEST_MODE_DEFAULT = false
  [ ] 12. Compilar (verificar sin errores)
  [ ] 13. Subir al Arduino
  [ ] 14. Verificar mensaje boot en LCD

FASE 4: PRUEBAS EN BANCO
  [ ] 15. Alimentar con fuente 12V (CON FUSIBLE)
  [ ] 16. Verificar LCD muestra "Hi Setterlee!"
  [ ] 17. Probar botón PAGE (cambia páginas)
  [ ] 18. Probar botón PEAK (muestra min/max)
  [ ] 19. Verificar sensor NTC lee temperatura ambiente
  [ ] 20. Calentar sensor → temperatura debe subir

FASE 5: INSTALACIÓN EN AUTO
  [ ] 21. Leer INSTALLATION_TIPS.md
  [ ] 22. Decidir ubicación componentes
  [ ] 23. Instalar sensor NTC en motor
  [ ] 24. Rutear cables (usar PINOUT.txt)
  [ ] 25. Conectar 12V (CON FUSIBLE, auto apagado)
  [ ] 26. Conectar serial a MS2
  [ ] 27. Verificar checklist pre-encendido

FASE 6: PRIMERA PRUEBA
  [ ] 28. Girar llave a ACC → verificar encendido
  [ ] 29. Girar llave a ON → verificar MS2 comunica
  [ ] 30. Arrancar motor → monitorear valores
  [ ] 31. Verificar temperatura aceite sube
  [ ] 32. Verificar presiones muestran valores reales
  [ ] 33. Test de manejo corto
  [ ] 34. Verificar todo funciona correctamente

FASE 7: AJUSTES FINALES
  [ ] 35. Activar alertas (ENABLE_ALERTS = true)
  [ ] 36. Personalizar páginas si es necesario
  [ ] 37. Ajustar rangos de alertas según tu motor
  [ ] 38. Verificar sistema estable después de 30min uso
  [ ] 39. Anotar valores normales de tu motor
  [ ] 40. ¡DISFRUTAR! 🏁

┌──────────────────────────────────────────────────────────────────────────────┐
│                       🔍 VERIFICACIONES IMPORTANTES                          │
└──────────────────────────────────────────────────────────────────────────────┘

ANTES DE SUBIR EL FIRMWARE:
  ⚠️  TEST_MODE_DEFAULT = false         (en config.h línea 219)
  ⚠️  LCD_ADDRESS = 0x27                (o 0x3F si tu LCD usa esa)
  ⚠️  MS2_BAUD_RATE = 9600              (verificar MS2 también)

ANTES DE CONECTAR 12V:
  ⚠️  Step-down ajustado a 5.0V         (medir con multímetro)
  ⚠️  Fusible 2A instalado              (en línea de 12V)
  ⚠️  No hay cortocircuitos             (verificar con multímetro)
  ⚠️  Resistencia 10K en circuito NTC   (verificar conexión)

ANTES DE ARRANCAR EL AUTO:
  ⚠️  Sensor NTC instalado correctamente (no toca chasis)
  ⚠️  Cables no rozan partes móviles    (motor, correas)
  ⚠️  MS2 serial conectado              (TX/RX correcto, GND común)
  ⚠️  Sistema funciona con llave ON     (sin arrancar primero)
